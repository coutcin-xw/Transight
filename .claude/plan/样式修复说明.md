

## 📁 关键配置文件

- `src/index.css` - TailwindCSS 导入和自定义样式
- `tailwind.config.js` - Tailwind 主题配置
- `postcss.config.js` - PostCSS 配置
- `vite.config.ts` - Vite 构建配置

## 🔧 如果样式仍然有问题

1. 清除缓存：
```bash
rm -rf node_modules dist
npm install
npm run build
```

2. 检查浏览器控制台是否有CSS加载错误

3. 确认 `dist/assets/*.css` 文件已生成且大小合理（约15KB）
