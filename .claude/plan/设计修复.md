# Transight 设计修复计划

## 发现的主要差异

### Translation Window
1. ❌ 缺少 Pin 按钮（置顶按钮）
2. ❌ 缺少 "原文" 标签
3. ❌ 缺少 Swap Button（交换语言按钮）
4. ❌ 结果显示错误 - 应该是多个翻译服务并排显示，不是单个列表
5. ❌ 缺少结果数量标签

### Settings Window
1. ❌ 侧边栏文字不对 - 应该是 "通用设置"、"快捷键设置" 等
2. ❌ 侧边栏图标缺失
3. ❌ 选中状态样式不对

### Service Config Window
1. ❌ 缺少服务列表选择

## 修复步骤
1. 修复 Translation Window 布局
2. 修复 Settings Window 侧边栏
3. 修复结果显示方式
