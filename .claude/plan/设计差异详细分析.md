# 设计稿详细对比分析

## Translation Window 差异

### 设计稿实际内容：
1. Title Bar:
   - Icon: 24x24, #3B82F6
   - Text: "Transight", fontSize:16, fontWeight:600
   - 3个按钮，每个 24x24

2. 没有 "原文" 标签！

3. Language Selector:
   - 源语言：90x30
   - 交换按钮：28x28 (arrow-right-left 图标)
   - 目标语言：90x30

4. Results Header:
   - "翻译结果" 标签
   - "3 个插件" 徽章（蓝色背景）

5. Results List:
   - 每个结果显示：
     - Plugin Icon (20x20, 不同颜色)
     - Plugin Name (fontSize:12, fontWeight:600)
     - Translation Result
     - Copy Button

## Settings Window 差异

### 侧边栏文字错误：
- ❌ "通用设置" → ✅ "常规设置"
- ❌ "快捷键设置" → ✅ "快捷键设置"
- ❌ "插件设置" → ✅ "服务设置"

### 缺少的设置项：
1. 翻译历史上限 (Setting Item 4 & 5)
2. Content Header 的 Icon (32x32)

### Service Tab 错误：
1. 侧边栏选中项错误（应该是第3项"服务设置"）
2. Service Header 图标缺失
3. 插件卡片布局错误

## Service Config Window 差异

1. Header:
   - Plugin Icon + 名称
   - 灰色占位图 (62x58, #CCCCCC)
   - 名称输入框
   - 描述文字

2. 表单字段：
   - API 密钥
   - 默认翻译语言（中文（简体））
   - 请求超时时间（30秒）

3. Footer Buttons:
   - "取消" (白色，灰色边框)
   - "保存配置" (蓝色，白色文字)
